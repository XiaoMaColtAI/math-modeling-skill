# 中英文术语对照表

| 符号 | 英文 | 中文 | 单位 |
|------|------|------|------|
| P | Producers | 生产者 | kg/m² |
| C | Crops | 作物生物量 | kg/m² |
| I | Insects | 昆虫种群数量 | individuals/m² |
| B | Birds | 鸟类种群数量 | individuals/km² |
| T | Bats | 蝙蝠种群数量 | individuals/km² |
| E | Earthworms | 蚯蚓生物量 | g/m² |
| H | Herbicides/Pesticides concentration | 化学物质浓度 | mg/kg |
| M | Habitat maturity | 栖息地成熟度 | 无量纲 [0,1] |
| t | Time | 时间 | day/year |
| r | Growth rate | 增长率 | 1/day |
| K | Carrying capacity | 环境容纳量 | 与物种相关 |
| a | Interaction coefficient | 相互作用系数 | 与关系相关 |
| d | Death rate | 死亡率 | 1/day |
| e | Energy conversion efficiency | 能量转化效率 | 无量纲 |
| δ | Degradation rate | 降解率 | 1/day |

| TL | Trophic level | 营养级 | 无量纲 |
| w | Weight | 权重 | 无量纲 |

## 生态学术语

| 英文 | 中文 | 说明 |
|------|------|------|
| Food web | 食物网 | 生态系统中物种间的营养关系网络 |
| Trophic level | 营养级 | 物种在食物链中的位置 |
| Producer | 生产者 | 能通过光合作用制造有机物的生物 |
| Consumer | 消费者 | 以其他生物为食的生物 |
| Primary consumer | 初级消费者 | 以植物为食的消费者 |
| Secondary consumer | 次级消费者 | 以初级消费者为食的生物 |
| Tertiary consumer | 三级消费者 | 以次级消费者为食的生物 |
| Keystone species | 关键种 | 对生态系统结构有重要影响的物种 |
| Ecosystem stability | 生态系统稳定性 | 生态系统抵抗干扰和恢复的能力 |
| Biodiversity | 生物多样性 | 生物种类的丰富程度 |
| Edge habitat | 边缘栖息地 | 不同生态系统交界处的栖息地 |
| Ecological succession | 生态演替 | 生态系统随时间变化的过程 |
| Carrying capacity | 环境容纳量 | 环境能支持的最大种群数量 |
| Biological control | 生物防治 | 利用天敌控制害虫 |
| Pollination | 授粉 | 花粉从雄蕊传递到雌蕊的过程 |
| Bioaccumulation | 生物富集 | 有害物质在生物体内积累 |
| Biomagnification | 生物放大 | 有害物质沿食物链逐级放大 |

## 农业术语

| 英文 | 中文 | 说明 |
|------|------|------|
| Monoculture | 单一栽培 | 只种植一种作物的农业系统 |
| Crop rotation | 轮作 | 不同作物轮流种植的农业实践 |
| Cover crops | 覆盖作物 | 为土壤保护而种植的作物 |
| Intercropping | 间作 | 同时种植多种作物 |
| Herbicide | 除草剂 | 用于杀灭杂草的化学物质 |
| Pesticide | 杀虫剂 | 用于杀灭害虫的化学物质 |
| Organic farming | 有机农业 | 不使用合成化学品的农业系统 |
| Conventional farming | 传统农业 | 使用合成化学品的农业系统 |
| Soil health | 土壤健康 | 土壤维持生态系统功能的能力 |
| Pest control | 害虫控制 | 管理害虫数量的措施 |
| Agricultural cycle | 农业周期 | 农业活动的周期性变化 |
| Seasonality | 季节性 | 随季节变化的规律 |
| Harvest | 收获 | 采集成熟作物的过程 |
| Planting | 种植 | 将作物种子或幼苗放入土壤的过程 |
