# 2025 ICM Problem E: Making Room for Agriculture - 题目分析报告

## 1. 问题背景与理解

### 1.1 问题背景

本题研究森林转变为农业用地后的生态系统演化过程。原本拥有丰富生物多样性的森林生态系统被清除后，被单一的农业种植系统所替代。随着时间推移，边缘栖息地逐渐成熟，原生物种开始重新出现，农业生态系统也随之发生变化。

### 1.2 核心问题

需要构建一个模型来追踪森林到农田的栖息地变化，包括：
- 自然过程：生态系统演化、物种重新出现、食物网动态
- 人类决策：农业实践、除草剂/杀虫剂使用、有机农业转型

## 2. 问题分解

### 2.1 自然过程建模

**问题1：建立当前农业生态系统模型**
- 构建食物网模型，包括生产者和消费者
- 考虑农业周期和季节性
- 分析除草剂和杀虫剂对生态系统的影响

**问题2：物种重新出现的建模**
- 模拟边缘栖息地成熟过程中物种的回归
- 纳入两个不同物种分析其影响

### 2.2 人类决策分析

**问题3：移除草剂后的生态系统稳定性**
- 分析移除草剂对生产者和消费者的稳定性影响

**问题4：引入蝙蝠恢复生态平衡**
- 将蝙蝠作为昆虫控制者和授粉者纳入模型
- 选择另一个有益物种进行比较分析

**问题5：有机农业分析**
- 分析不同有机农业组合的影响
- 评估 pest control、crop health、plant reproduction、biodiversity、long-term sustainability、cost effectiveness

### 2.3 成果输出

- 一页给农民的信关于有机农业实践
- PDF论文（不超过25页）

## 3. 模型选择与设计

### 3.1 核心数学模型：改进的Lotka-Volterra生态系统动力学模型

本问题的核心是建立描述农业生态系统中物种间相互作用的微分方程模型。

#### 3.1.1 经典Lotka-Volterra方程

```
dx/dt = αx - βxy
dy/dt = δxy - γy
```

其中：
- x: 食饵种群数量
- y: 捕食者种群数量
- α: 食饵自然增长率
- β: 捕食系数
- δ: 捕食者转化效率
- γ: 捕食者自然死亡率

#### 3.1.2 改进的多元生态系统模型

为适应本题需求，建立包含多个营养级的生态系统动力学模型：

**模型框架：**

```
dP/dt = f(P, C, H, t, E)
dC/dt = g(P, C, H, t, E)
dH/dt = h(P, C, H, t, E)
```

其中：
- P: 生产者（作物、植物）向量
- C: 消费者（昆虫、鸟类、蝙蝠等）向量
- H: 人类活动因素（除草剂、杀虫剂使用等）
- t: 时间（考虑季节性）
- E: 环境因子

### 3.2 食物网模型：有向加权图

#### 3.2.1 图论表示

使用有向图 G = (V, E) 表示食物网：
- 顶点集 V: 生态系统中的所有物种
- 边集 E: 物种间的捕食/竞争关系
- 权重 w_ij: 物种j对物种i的影响强度

#### 3.2.2 营养级分析

```
TL_i = 1 + Σ(TL_j × w_ij / Σw_ij)
```

其中 TL_i 为物种i的营养级。

### 3.3 季节性模型：周期函数

```
S(t) = A·sin(2πt/T) + B
```

其中：
- T: 周期（一年）
- A: 波动幅度
- B: 基准值

### 3.4 除草剂/杀虫剂影响模型

#### 3.4.1 直接效应

```
dC_i/dt = -k_chem × C_i × Chem(t)
```

其中 Chem(t) 为化学物质浓度。

#### 3.4.2 间接效应（生物富集）

```
Chem_predator(n) = Chem_predator(n-1) + EF × Chem_prey × consumption_rate
```

### 3.5 生态系统稳定性分析

#### 3.5.1 局部稳定性：Jacobian矩阵特征值

计算平衡点的 Jacobian 矩阵 J，分析其特征值 λ：

```
J = [∂f_i/∂x_j]
```

- 若所有 Re(λ_i) < 0，则系统局部稳定
- 若存在 Re(λ_i) > 0，则系统不稳定

#### 3.5.2 鲁棒性指标

```
R = 1 / (σ_max(J))
```

其中 σ_max 为最大奇异值。

### 3.6 有机农业综合评价模型

使用层次分析法（AHP）+ 熵权法确定权重，TOPSIS方法进行综合评价。

**指标体系：**
| 一级指标 | 二级指标 |
|---------|---------|
| 生态效益 | 生物多样性、土壤健康、生态平衡度 |
| 经济效益 | 产量、成本、利润率 |
| 社会效益 | 可持续性、抗风险能力 |

## 4. 具体模型公式

### 4.1 基础农业生态系统模型

```
dC/dt = r_c·C·(1 - C/K_c) - a_CI·C·I - S_C(t)·C
dI/dt = e·a_CI·C·I - d_I·I - a_IB·I·B - S_I(t)·I
dB/dt = e·a_IB·I·B - d_B·B - S_B(t)·B
```

其中：
- C: 作物生物量
- I: 昆虫种群数量
- B: 鸟类种群数量
- S_x(t): 季节性函数
- r_c: 作物增长率
- K_c: 作物环境容纳量
- a_CI: 昆虫对作物的影响系数
- e: 能量转化效率
- d_I, d_B: 昆虫和鸟类死亡率

### 4.2 加入除草剂/杀虫剂的模型

```
dC/dt = r_c·C·(1 - C/K_c) - a_CI·C·I + f_H(C, H) - S_C(t)·C
dI/dt = e·a_CI·C·I - d_I·I - a_IB·I·B - g_H(I, H) - S_I(t)·I
dB/dt = e·a_IB·I·B - d_B·B - h_H(B, H) - S_B(t)·B
dH/dt = Input(t) - δ·H
```

其中：
- H: 化学物质浓度
- f_H: 化学物质对作物的影响（可能是正面的除草效应）
- g_H: 化学物质对昆虫的影响（杀虫效应）
- h_H: 化学物质对鸟类的影响（负面生物富集效应）
- Input(t): 化学物质投放
- δ: 化学物质降解率

### 4.3 加入蝙蝠的扩展模型

```
dC/dt = r_c·C·(1 - C/K_c) - a_CI·C·I + f_H(C, H) + p_B·B - S_C(t)·C
dI/dt = e·a_CI·C·I - d_I·I - a_IB·I·B - a_IT·I·T - g_H(I, H) - S_I(t)·I
dB/dt = e·a_IB·I·B - d_B·B - h_H(B, H) - S_B(t)·B
dT/dt = e·a_IT·I·T + e·p_B·C·T - d_T·T - S_T(t)·T
dH/dt = Input(t) - δ·H
```

其中：
- T: 蝙蝠种群数量
- a_IT: 蝙蝠对昆虫的捕食系数
- p_B: 蝙蝠授粉对作物增长的贡献
- S_T(t): 蝙蝠的季节性（迁徙行为）

### 4.4 另一个有益物种选择

**选择：蚯蚓（Earthworms）**

理由：
1. 土壤改良者，改善土壤结构和肥力
2. 有机物分解者，促进营养循环
3. 与蝙蝠形成不同生态位的互补效应

```
dE/dt = r_E·E·(1 - E/K_E·H_organic) - d_E·E
dC/dt = ... + s_E·E  # 蚯蚓对作物生长的促进作用
```

其中：
- E: 蚯蚓生物量
- H_organic: 土壤有机质含量
- s_E: 蚯蚓对作物的影响系数

### 4.5 物种重新出现模型

```
dS_i/dt = r_i·S_i·(1 - S_i/K_i) + im_i(t) - em_i(t)
```

其中：
- S_i: 物种i的种群数量
- im_i(t): 迁入率（随栖息地成熟度增加）
- em_i(t): 迁出率

栖息地成熟度模型：
```
dM/dt = α·M·(1 - M) + β·(1 - H_intensity)
```

其中 M ∈ [0,1] 为栖息地成熟度。

### 4.6 有机农业评价模型

#### 4.6.1 AHP权重确定

构造判断矩阵 A = (a_ij)，计算权重向量 w：
```
w_i = (∏a_ij)^(1/n) / Σ(∏a_kj)^(1/n)
```

#### 4.6.2 TOPSIS综合评分

```
D+_i = √Σ(w_j × (x_ij - x+_j)^2)
D-_i = √Σ(w_j × (x_ij - x-_j)^2)
C_i = D-_i / (D+_i + D-_i)
```

## 6. 创新点

1. **多层次耦合模型**: 将食物网结构、季节性、人类活动耦合到统一的动力学框架

2. **动态栖息地成熟度模型**: 将栖息地质量变化纳入物种重新出现的模拟

3. **多维度有机农业评价**: 结合生态、经济、社会三方面的综合评估

4. **蝙蝠-蚯蚓协同效应模型**: 分析不同生态位有益物种的互补作用
