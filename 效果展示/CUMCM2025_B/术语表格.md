# 术语表 (Terminology)

本文档用于确保数学建模全过程的术语一致性，便于论文撰写和英文化。

## 八、物理公式汇总

### 8.1 斯涅尔定律（折射定律）

```
n₀·sin(θ₀) = n₁·sin(θ₁)
```

### 8.2 光程差

```
Δ = 2·n₁·d·cos(θ₁)
```

### 8.3 相位差

```
δ = 2π·Δ/λ = 4π·n₁·d·cos(θ₁)/λ
```

### 8.4 干涉条件

- 相长干涉：`Δ = m·λ` 或 `δ = 2mπ`
- 相消干涉：`Δ = (m+0.5)·λ` 或 `δ = (2m+1)π`

### 8.5 双光束干涉反射率

```
R = (R₁ + R₂ + 2√(R₁R₂)·cos(δ)) / (1 + R₁R₂ + 2√(R₁R₂)·cos(δ))
```

### 8.6 多光束干涉反射率（爱里函数）

```
R = R₁ + (1-R₁)²·R₂ / (1 + R₁R₂ - 2√(R₁R₂)·cos(δ))
```

### 8.7 厚度计算公式（极值点法）

```
d = 1/(2·n₁·Δν·cos(θ₁))
```

### 8.8 厚度计算公式（频谱法）

```
d = 1/(2·n₁·f·cos(θ₁))
```

## 十、注意事项

1. **符号一致性**：全文使用统一的数学符号表示同一物理量
2. **单位规范**：在公式中标明单位，计算时统一单位制
3. **中英文对照**：关键术语首次出现时标注英文
4. **图表标注**：图表中的术语使用标准符号
5. **公式编号**：重要公式统一编号，便于引用
