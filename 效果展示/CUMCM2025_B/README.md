# 碳化硅外延层厚度的确定 - 2025年全国大学生数学建模竞赛B题

## 项目概述

本项目针对2025年全国大学生数学建模竞赛B题"碳化硅外延层厚度的确定"建立了完整的数学模型和求解算法。红外干涉法是一种非破坏性的外延层厚度测量方法，通过分析红外光在外延层界面产生的干涉条纹来确定外延层厚度。

本项目基于薄膜光学理论建立了干涉条纹分析模型，设计了基于快速傅里叶变换（FFT）的厚度计算算法，并对碳化硅和硅晶圆片的实测数据进行了分析计算。

## 使用的数学模型

### 1. 单次反射干涉模型（问题1）

基于菲涅尔方程和薄膜光学理论，建立了双光束干涉的反射率公式：

$$R = \frac{R_1 + R_2 + 2\sqrt{R_1R_2}\cos\delta}{1 + R_1R_2 + 2\sqrt{R_1R_2}\cos\delta}$$

其中相位差为：

$$\delta = \frac{4\pi n_1 d\cos\theta_1}{\lambda}$$

厚度计算公式：

$$d = \frac{1}{2n_1\Delta\nu\cos\theta_1}$$

### 2. FFT频谱分析法（问题2）

采用快速傅里叶变换方法从光谱数据中提取干涉条纹的周期信息：
- 数据预处理（平滑滤波、去趋势）
- FFT变换获取频谱
- 峰值检测确定主频率
- 根据主频率计算厚度

### 3. 多光束干涉模型（问题3）

引入精锐系数判断多光束干涉的显著性：

$$F = \frac{4\sqrt{R_1R_2}}{(1-R_1R_2)^2}$$

- 碳化硅：$R \approx 5\%$, $F = 0.201$（双光束模型已足够精确）
- 硅：$R \approx 30\%$, $F = 1.449$（需要使用爱里函数模型修正）

## 核心算法

### ThicknessCalculator 类

厚度计算器类，实现了以下核心方法：

- `preprocess_data()`: 数据预处理（平滑滤波）
- `find_extrema()`: 极值点检测
- `calculate_thickness_extrema()`: 极值点法计算厚度
- `calculate_thickness_fft()`: FFT频谱分析法计算厚度
- `calculate_thickness_fitting()`: 全谱拟合法计算厚度

### 关键参数

| 材料 | 折射率 | 界面反射率 | 精锐系数 |
|------|--------|------------|----------|
| 碳化硅 (SiC) | 2.65 | ~5% | 0.201 |
| 硅 (Si) | 3.45 | ~30% | 1.449 |

## 可靠性分析

### 不同入射角结果一致性

- 碳化硅：10°与15°结果相对偏差 = **0.26%**
- 硅：10°与15°结果相对偏差 = **0.15%**

### 误差来源

1. 折射率参数误差（色散、掺杂浓度变化）
2. FFT频率分辨率限制
3. 多光束干涉影响（对硅晶圆片影响较大）
4. 测量系统误差（波数定标、入射角等）

## 创新点

1. **多算法融合**：结合极值点法、频谱分析法、全谱拟合法，提高计算精度
2. **多光束干涉修正**：建立完整的多光束干涉修正模型
3. **双角度验证**：利用不同入射角的结果交叉验证
4. **误差量化分析**：系统分析各因素对计算精度的影响
