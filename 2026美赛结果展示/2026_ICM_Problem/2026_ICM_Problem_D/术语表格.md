# 术语表 (Terminology)

本文档用于确保数学建模全过程的术语一致性，便于论文撰写和英文化。

## 一、核心概念术语

| 中文名称 | 英文名称 | 缩写符号 | 定义/说明 |
|---------|---------|---------|----------|
| 动态决策模型 | Dynamic Decision-Making Model | DDM | 根据状态变化调整策略的优化模型 |
| 多目标优化 | Multi-Objective Optimization | MOO | 同时优化多个冲突目标的优化问题 |
| 球员价值评估 | Player Valuation | PV | 综合评估球员对球队贡献的量化指标 |
| 薪资帽 | Salary Cap | SC | 联盟规定的球队总薪资上限 |
| 帕累托前沿 | Pareto Frontier | PF | 多目标优化中非支配解的集合 |
| 风险价值 | Value at Risk | VaR | 在给定置信水平下的最大可能损失 |
| 条件风险价值 | Conditional Value at Risk | CVaR | 超过VaR的平均损失 |
| 马尔科夫决策过程 | Markov Decision Process | MDP | 序列决策的数学框架 |
| 球队价值 | Team Value | TV | 球队的市场价值，包括品牌、资产等 |
| 竞技表现 | Athletic Performance | AP | 球队在比赛中的表现指标（胜率等） |
| 联盟扩张 | League Expansion | LE | 联盟增加新球队的过程 |

**使用说明**：
