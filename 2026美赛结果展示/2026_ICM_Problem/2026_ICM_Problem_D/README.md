# 2026 ICM Problem D: Managing Sports for Success

## 项目概述

本项目为2026年ICM Problem D（体育球队商业管理）的完整数学建模解决方案。项目建立了一个综合的动态决策模型，帮助职业体育球队的老板和总经理在竞技成功和财务健康之间取得平衡。

### 研究对象

- **联赛**: WNBA（美国女子职业篮球联赛）
- **球队**: Las Vegas Aces（拉斯维加斯王牌队）
- **目标**: 最大化球队利润和球队价值

## 使用的数学模型

| 模型 | 用途 | 方法 |
|------|------|------|
| AHP层次分析法 | 确定评价指标主观权重 | 特征值分解 |
| 熵权法 | 确定评价指标客观权重 | 信息熵计算 |
| TOPSIS | 球员综合价值排序 | 优劣解距离法 |
| 多目标优化 | 阵容优化（利润 vs 表现） | 差分进化算法 |
| 蒙特卡洛模拟 | 伤病风险分析 | 随机模拟 |
| 需求函数优化 | 票务定价 | 网格搜索 |

## 建模流程

1. **问题分析**: 分析体育商业管理的核心问题
2. **数据收集**: 获取球员数据、球队数据和财务数据
3. **球员价值评估**: 使用AHP-熵权-TOPSIS组合模型
4. **阵容优化**: 多目标优化求解帕累托前沿
5. **票务定价**: 不同场景下的最优定价策略
6. **风险分析**: 蒙特卡洛模拟伤病影响
7. **联盟扩张**: 分析扩张对各球队的影响
8. **策略建议**: 综合分析和建议

## 管理策略建议

### 球员获取策略

1. 优先考虑综合价值得分高的球员
2. 平衡薪资和表现，追求性价比
3. 适当投资明星球员以提升商业价值
4. 保持合理的年龄结构

### 票务定价策略

1. 根据对手实力动态调整票价
2. 周末和强对手比赛可适当提高票价
3. 弱对手比赛可降低票价以提高上座率
4. 考虑季票优惠培养忠实球迷

### 风险管理策略

1. 建立伤病应急基金
2. 保持阵容深度以应对伤病
3. 考虑购买相关保险
4. 监控关键球员伤病风险

### 联盟扩张应对

1. 提前锁定关键球员合同
2. 关注新球队所在市场的薪资影响
3. 利用扩张机会吸引新球迷
