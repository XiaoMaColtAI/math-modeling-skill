# 术语表 (Terminology)

本文档用于确保数学建模全过程的术语一致性，便于论文撰写和英文化。

## 一、核心概念术语

| 中文名称 | 英文名称 | 缩写符号 | 定义/说明 |
|---------|---------|---------|----------|
| 被动式太阳能遮阳 | Passive Solar Shading | PSS | 利用建筑设计和遮阳设施控制太阳辐射，减少供暖制冷能耗的技术 |
| 太阳高度角 | Solar Altitude Angle | $\alpha$ | 太阳光线与地平线的夹角，范围0°-90° |
| 太阳方位角 | Solar Azimuth Angle | $\gamma$ | 太阳光线在地平面上的投影与正南方向的夹角 |
| 太阳赤纬角 | Solar Declination Angle | $\delta$ | 太阳光线与地球赤道面的夹角，随季节变化 |
| 时角 | Hour Angle | $H$ | 由于地球自转，太阳位置相对于当地子午线的角度 |
| 直射辐射 | Direct/Beam Radiation | $I_{beam}$ | 直接来自太阳未被散射的太阳辐射 |
| 散射辐射 | Diffuse Radiation | $I_{diffuse}$ | 被大气散射后到达地面的太阳辐射 |
| 反射辐射 | Reflected Radiation | $I_{reflected}$ | 被地面反射的太阳辐射 |
| 太阳热增益系数 | Solar Heat Gain Coefficient | SHGC | 窗户系统允许的太阳辐射热量比例 |
| 遮阳系数 | Shading Coefficient | $F_{shading}$ 或 $SC$ | 遮阳设施阻挡太阳辐射的比例，0-1 |
| 窗墙比 | Window-to-Wall Ratio | WWR | 窗户面积占外墙面积的比例 |
| 悬挑遮阳 | Overhang Shading | - | 水平突出的遮阳设施 |
| 百叶遮阳 | Louver Shading | - | 由可调角度叶片组成的遮阳系统 |
| 遮蔽设施 | Brise-soleil | - | 建筑外部的遮阳系统，源自法语 |
| 热质量 | Thermal Mass | $C_{eff}$ | 建筑材料存储和释放热量的能力 |
| 热阻 | Thermal Resistance | R-value | 材料抵抗热传递的能力 |
| 制冷负荷 | Cooling Load | $Q_c$ | 移除建筑内多余热量所需的能量 |
| 供暖负荷 | Heating Load | $Q_h$ | 补充建筑内热量损失所需的能量 |
| 度日 | Degree Day | DD | 温度与基准温度差异的累积量 |
| 制冷度日 | Cooling Degree Day | CDD | 日均温度高于制冷基准的度数累积 |
| 供暖度日 | Heating Degree Day | HDD | 日均温度低于供暖基准的度数累积 |
| 净零制冷 | Net-zero Cooling | NZC | 制冷能耗完全由可再生能源抵消 |
| 太阳正午 | Solar Noon | - | 太阳达到最高点的时刻 |
| 夏至 | Summer Solstice | - | 北半球白昼最长的一天（约6月21日） |
| 冬至 | Winter Solstice | - | 北半球白昼最短的一天（约12月21日） |
| 春/秋分 | Vernal/Autumnal Equinox | - | 昼夜平分的日子（约3月21日和9月23日） |
| 典型气象年 | Typical Meteorological Year | TMY | 代表长期气候特征的典型年气象数据 |
| Pareto最优 | Pareto Optimal | - | 多目标优化中无法再改善某一目标而不损害其他目标的状态 |
| Pareto前沿 | Pareto Front | - | 多目标优化中所有Pareto最优解的集合 |

## 五、评价指标术语

| 中文指标 | 英文指标 | 公式/定义 |
|---------|---------|----------|
| 制冷负荷减少率 | Cooling Load Reduction Rate | $\eta_c = \frac{E_{c0} - E_{c1}}{E_{c0}} \times 100\%$ |
| 供暖负荷减少率 | Heating Load Reduction Rate | $\eta_h = \frac{E_{h0} - E_{h1}}{E_{h0}} \times 100\%$ |
| 全年总能耗 | Annual Total Energy Consumption | $E_{total} = E_{cooling} + E_{heating}$ |
| 净零制冷指数 | Net-zero Cooling Index | $I_{NZC} = 1 - \frac{E_{cooling}}{E_{solar}}$ |
| 能效比 | Energy Efficiency Ratio | $EER = \frac{Q_{cooling}}{E_{input}}$ |
| 太阳能得热比 | Solar Heat Gain Ratio | $SHGR = \frac{Q_{solar}}{Q_{total}}$ |
| 遮阳效率 | Shading Efficiency | $\eta_s = \frac{E_{reduced}}{E_{incident}}$ |
| 舒适度指标 | Predicted Percentage Dissatisfied | PPD（越低越好） |
| 日照利用系数 | Daylight Utilization Factor | DUF |
| 投资回收期 | Payback Period | $PBP = \frac{C_{investment}}{S_{annual}}$ |

## 术语使用规范

### 英文论文翻译对照

1. **被动式太阳能遮阳**: "Passive Solar Shading" 或 "Passive Solar Shading Systems"
2. **遮阳设施**: "Shading Devices" 或 "Solar Shading Systems"
3. **悬挑**: "Overhang" 或 "Horizontal Shading Device"
4. **百叶**: "Louvers" 或 "Louvered Shading"
5. **太阳热增益**: "Solar Heat Gain"
6. **热质量**: "Thermal Mass"
7. **窗墙比**: "Window-to-Wall Ratio" (WWR)
8. **制冷负荷**: "Cooling Load"
9. **供暖负荷**: "Heating Load"
10. **净零制冷**: "Net-zero Cooling"
11. **多目标优化**: "Multi-objective Optimization"
12. **Pareto最优**: "Pareto Optimality"

### 符号规范

1. 角度使用希腊字母：$\alpha, \gamma, \delta, \phi, \theta$
2. 负荷使用Q加下标：$Q_{solar}, Q_{cooling}, Q_{heating}$
3. 能耗使用E加下标：$E_{total}, E_{cooling}, E_{heating}$
4. 系数使用F或SC：$F_{shading}, SC$
5. 材料属性使用$\rho, \epsilon$等
