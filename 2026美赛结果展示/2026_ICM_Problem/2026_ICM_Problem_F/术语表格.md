# 术语表 (Terminology)

本文档用于确保数学建模全过程的术语一致性，便于论文撰写和英文化。

---

## 一、核心概念术语

| 中文名称 | 英文名称 | 缩写符号 | 定义/说明 |
|---------|---------|---------|----------|
| 生成式人工智能 | Generative Artificial Intelligence | Gen-AI | 能够生成新内容（文本、图像、代码等）的人工智能技术 |
| 灰色预测模型 | Grey Prediction Model | GM(1,1) | 用于小样本、贫信息系统预测的灰色系统理论模型 |
| 层次分析法 | Analytic Hierarchy Process | AHP | 一种定性与定量相结合的多准则决策方法 |
| 熵权法 | Entropy Weight Method | EWM | 基于信息熵的客观赋权方法 |
| 优劣解距离法 | Technique for Order Preference by Similarity to Ideal Solution | TOPSIS | 通过计算与理想解距离进行方案排序的方法 |
| 系统动力学 | System Dynamics | SD | 研究复杂系统动态行为的建模方法 |
| 美国劳工统计局 | Bureau of Labor Statistics | BLS | 美国联邦政府统计机构，提供就业数据 |
| 职业就业统计 | Occupational Employment Statistics | OES | BLS发布的职业就业和工资数据 |
| 职业展望手册 | Occupational Outlook Handbook | OOH | BLS发布的职业发展预测报告 |
| 后验差检验 | Posterior Difference Test | - | 灰色预测模型的精度检验方法 |

## 二、数学符号术语

| 符号 | 含义 | 单位 | 英文名称 |
|-----|------|------|----------|
| $X^{(0)}$ | 原始数据序列 | - | Original data sequence |

| $a$ | 灰色预测发展系数 | - | Development coefficient |
| $b$ | 灰色预测灰色作用量 | - | Grey action quantity |
| $\lambda_{\max}$ | 最大特征值 | - | Maximum eigenvalue |

| $CR$ | 一致性比例 | - | Consistency Ratio |
| $RI$ | 随机一致性指标 | - | Random Consistency Index |
| $e_j$ | 第j个指标的信息熵 | - | Information entropy of indicator j |
| $w_j$ | 第j个指标的权重 | - | Weight of indicator j |
| $D^+$ | 到正理想解的距离 | - | Distance to positive ideal solution |
| $D^-$ | 到负理想解的距离 | - | Distance to negative ideal solution |
| $C_i$ | 相对贴近度 | - | Relative closeness coefficient |

| $S_1$ | 原始数据标准差 | - | Standard deviation of original data |
| $S_2$ | 残差标准差 | - | Standard deviation of residuals |

## 五、评价指标术语

| 中文指标 | 英文指标 | 公式/定义 |
|---------|---------|----------|
| 平均相对误差 | Mean Relative Error | $MRE = \frac{1}{n}\sum_{i=1}^{n}|\frac{y_i - \hat{y}_i}{y_i}|$ |
| 均方根误差 | Root Mean Square Error | $RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}$ |
| 后验差比值 | Posterior Difference Ratio | $C = \frac{S_2}{S_1}$ |
| 小误差概率 | Small Error Probability | $P = P(|e(k)| < 0.6745S_1)$ |
| 关联度 | Grey Relational Grade | - |
| 分辨系数 | Distinguishing Coefficient | $\rho$，通常取0.5 |
| 信息效用值 | Information Utility Value | $d_j = 1 - e_j$ |
| 贴近度 | Closeness Coefficient | $C_i = \frac{D_i^-}{D_i^+ + D_i^-}$ |
