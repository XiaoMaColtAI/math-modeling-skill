## 一、核心概念术语

| 中文名称 | 英文名称 | 缩写符号 | 定义/说明 |
|---------|---------|---------|----------|
| 与星共舞 | Dancing with the Stars | DWTS | 美国舞蹈比赛电视节目 |
| 名人/明星 | Celebrity | - | 参加比赛的非专业舞者 |
| 专业舞者 | Professional Dancer | Pro Dancer | 与名人配对的专业舞蹈演员 |
| 评委分数 | Judge Score | S | 评委给选手打出的分数（1-10分） |
| 粉丝投票 | Fan Vote | V | 观众对选手的投票数 |
| 综合分数 | Combined Score | C | 评委分数和粉丝投票合并后的分数 |
| 淘汰 | Elimination | - | 每周综合分数最低的选手被淘汰 |
| 排名合并法 | Rank-based Method | - | 将分数和投票都转换为排名后相加 |
| 百分比合并法 | Percentage-based Method | - | 将分数和投票都转换为百分比后相加 |
| 底部两位裁决 | Bottom Two Judge Elimination | B2JE | 评委从底部两位中选一人淘汰 |
| 争议案例 | Controversial Case | - | 评委分数和粉丝投票差异巨大的情况 |

## 二、数学符号术语

### 2.1 索引符号

| 符号 | 含义 | 单位 | 英文名称 |
|-----|------|------|----------|
| $i$ | 第i个选手/名人 | - | contestant i / celebrity i |
| $j$ | 第j个评委 | - | judge j |
| $w$ | 第w周 | - | week w |
| $s$ | 第s赛季 | - | season s |
| $p$ | 第p个专业舞者 | - | professional dancer p |

### 2.2 数据变量

| 符号 | 含义 | 单位 | 英文名称 |
|-----|------|------|----------|
| $S_{i,w}$ | 选手i在第w周的评委总分 | 分 | total judge score for contestant i in week w |
| $S_{i,w,j}$ | 评委j给选手i在第w周的分数 | 分 | judge j's score for contestant i in week w |
| $V_{i,w}$ | 选手i在第w周的粉丝投票数 | 票 | fan votes for contestant i in week w |
| $C_{i,w}$ | 选手i在第w周的综合分数 | - | combined score for contestant i in week w |
| $A_{i,w}$ | 选手i在第w周的平均评委分数 | 分 | average judge score for contestant i in week w |
| $R_{i,w}$ | 选手i在第w周的最终排名 | - | final placement of contestant i in season |

### 2.3 排名和百分比变量

| 符号 | 含义 | 单位 | 英文名称 |
|-----|------|------|----------|
| $\text{JRank}_{i,w}$ | 选手i在第w周的评委排名 | - | judge rank of contestant i in week w |
| $\text{FRank}_{i,w}$ | 选手i在第w周的粉丝投票排名 | - | fan vote rank of contestant i in week w |
| $\text{JPct}_{i,w}$ | 选手i在第w周的评委分数百分比 | % | judge score percentage of contestant i in week w |
| $\text{FPct}_{i,w}$ | 选手i在第w周的粉丝投票百分比 | % | fan vote percentage of contestant i in week w |

### 2.4 特征变量

| 符号 | 含义 | 单位 | 英文名称 |
|-----|------|------|----------|
| $\text{Age}_i$ | 选手i的年龄 | 岁 | age of contestant i |
| $\text{Industry}_i$ | 选手i的行业类别 | - | industry category of contestant i |
| $\text{Partner}_i$ | 选手i的专业舞者伙伴 | - | professional partner of contestant i |
| $\text{State}_i$ | 选手i的家乡州 | - | home state of contestant i |
| $\text{Country}_i$ | 选手i的家乡国家 | - | home country of contestant i |

### 2.5 回归系数

| 符号 | 含义 | 英文名称 |
|-----|------|----------|
| $\beta_0, \beta_1, \ldots$ | 评委分数模型回归系数 | regression coefficients for judge score model |
| $\alpha_0, \alpha_1, \ldots$ | 粉丝投票模型回归系数 | regression coefficients for fan vote model |
| $\gamma_0, \gamma_1, \ldots$ | 最终排名模型回归系数 | regression coefficients for placement model |
| $\theta_k$ | 有序回归的阈值参数 | threshold parameter for ordinal regression |
| $w_1, w_2, w_3$ | 新系统的权重参数 | weight parameters for new system |

### 2.6 集合符号

| 符号 | 含义 | 英文名称 |
|-----|------|----------|
| $A_w$ | 第w周参赛选手集合 | set of contestants in week w |
| $E_w$ | 第w周被淘汰选手集合 | set of eliminated contestants in week w |
| $S_s$ | 第s赛季所有选手集合 | set of all contestants in season s |

### 2.7 统计量符号

| 符号 | 含义 | 英文名称 |
|-----|------|----------|
| $\mu$ | 均值 | mean |
| $\sigma$ | 标准差 | standard deviation |
| $\rho$ | 相关系数 | correlation coefficient |
| $R^2$ | 决定系数 | coefficient of determination |
| $\epsilon$ | 误差项 | error term |

## 五、评价指标术语

### 5.1 估计精度指标

| 中文指标 | 英文指标 | 符号/公式 | 说明 |
|---------|---------|----------|------|
| 均方误差 | Mean Squared Error | MSE = $\frac{1}{n}\sum_{i=1}^n(y_i-\hat{y}_i)^2$ | 预测误差平方的平均 |
| 均方根误差 | Root Mean Squared Error | RMSE = $\sqrt{\text{MSE}}$ | MSE的平方根 |

| 一致性准确率 | Consistency Accuracy | - | 估计与实际一致的比例 |
| Kappa系数 | Cohen's Kappa | $\kappa$ | 衡量评价者间一致性 |
| Spearman相关 | Spearman Correlation | $\rho_s$ | 排名相关性 |

### 5.3 不确定性指标

### 5.4 模型选择指标
