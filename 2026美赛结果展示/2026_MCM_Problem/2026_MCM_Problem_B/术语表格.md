# 术语表 (Terminology)

本文档用于确保2026 MCM Problem B数学建模全过程的术语一致性，便于论文撰写和英文化。

## 一、核心概念术语

| 中文名称 | 英文名称 | 缩写符号 | 定义/说明 |
|---------|---------|---------|----------|
| 太空电梯系统 | Space Elevator System | SES | 由系索、地球港口、顶点锚组成的运输系统 |
| 银河港口 | Galactic Harbour | GH | 包含地球港口、两个系索和两个顶点锚的设施 |
| 地球港口 | Earth Port | EP | 地球表面的支持设施 |
| 系索 | Tether | - | 长度100,000km的石墨烯材料连接地球与顶点锚 |
| 顶点锚 | Apex Anchor | AA | 系索末端的太空配重 |
| 地球同步轨道 | Geosynchronous Orbit | GEO | 距地面约35,786km的轨道 |
| 月球殖民地 | Moon Colony | MC | 可容纳100,000人的月球栖息地 |
| 月球殖民地管理署 | Moon Colony Management Agency | MCM | 负责建设月球殖民地的虚构机构 |
| 传统火箭 | Traditional Rocket | - | 从地球发射场直接飞往月球的运载工具 |
| 发射场 | Launch Site | - | 地球上用于发射火箭的设施 |

## 二、数学符号术语

| 符号 | 含义 | 单位 | 英文名称 |
|-----|------|------|----------|
| $Q_{total}$ | 建筑材料总需求 | 公吨 | total material demand |
| $Q_e$ | 太空电梯年运输量 | 公吨/年 | elevator annual capacity |
| $Q_r$ | 火箭单次载重量 | 公吨/次 | rocket payload capacity |
| $C_e$ | 太空电梯单位运输成本 | $/吨 | elevator unit cost |
| $C_r$ | 火箭单位运输成本 | $/吨 | rocket unit cost |
| $N_{port}$ | 银河港口数量 | 个 | number of galactic harbours |
| $N_{site}$ | 发射场数量 | 个 | number of launch sites |
| $f_i$ | 发射场i的年发射频率 | 次/年 | launch frequency |
| $T$ | 项目完成时间 | 年 | project completion time |
| $Z$ | 总成本 | 美元 | total cost |
| $\eta_e$ | 太空电梯可用率 | - | elevator availability rate |
| $\eta_r$ | 火箭成功率 | - | rocket success rate |
| $x_{e,t}$ | 第t年电梯运输量 | 公吨 | elevator shipment in year t |
| $x_{r,i,t}$ | 第t年从发射场i的火箭运输量 | 公吨 | rocket shipment from site i in year t |
| $y_{i,t}$ | 是否使用发射场i的二元变量 | 0/1 | binary variable for using launch site i |
| $W_{total}$ | 年用水总量 | 公吨 | annual water demand |
| $E_{CO2}$ | 碳排放量 | 吨CO2 | carbon emissions |

## 五、评价指标术语

| 中文指标 | 英文指标 | 公式/定义 |
|---------|---------|----------|
| 总成本 | Total Cost | $Z = \sum C \cdot Q$ |
| 完成时间 | Completion Time | $T = \min \{t: \sum_{i=1}^t Q_i \geq Q_{total}\}$ |
| 单位运输成本 | Unit Transportation Cost | $C_{unit} = Z / Q_{total}$ |
| 成本-效益比 | Cost-Benefit Ratio | CBR = Cost / Benefit |
| 方案成功率 | Success Probability | $P(\text{完成时间} \leq T_{target})$ |
| 碳排放量 | Carbon Emissions | $E = \sum \gamma_{CO2} \cdot N$ |
| 环境影响指数 | Environmental Impact Index | EII，综合评估指标 |
| 系统利用率 | Utilization Rate | $\rho = \lambda / (c \cdot \mu)$ |
| 平均等待时间 | Average Waiting Time | $W_q$ |
| 鲁棒性指数 | Robustness Index | RI，衡量方案抗干扰能力 |
