# 2026 MCM Problem B - 使用太空电梯系统建造月球殖民地

## 项目概述

本项目针对2026年美国大学生数学建模竞赛（MCM）B题，建立了用于规划月球殖民地材料运输的综合数学模型。通过混合整数线性规划、蒙特卡洛模拟等方法，比较了太空电梯系统与传统火箭运输的成本、时间、可靠性和环境影响。

### 推荐方案：混合运输策略

**运输分配**：
- 太空电梯承担约68%的材料运输（12.5百万吨）
- 火箭承担约32%的运输量（5.8百万吨）
- 在35年内完成1亿公吨材料运输
- 总成本约435亿美元

**技术进步假设**：
- 火箭发射频率年增长率：5%
- 火箭运输成本年下降率：5%
- 太空电梯建设周期：10年

### 敏感性分析结果

10,000次蒙特卡洛模拟结果（混合方案）：
- **完成时间**：95%置信区间为31-39年，均值35.2年
- **总成本**：95%置信区间为371-503亿美元，均值437亿美元
- **碳排放**：95%置信区间为3.0-4.3千吨，均值3.6千吨

### 水资源需求

- **年用水量**：328,500吨（10万人）
- **推荐运输**：90%太空电梯 + 10%火箭
- **年运输成本**：约4.6亿美元

## 模型方法

1. **混合整数线性规划（MILP）**：优化运输组合，优先使用成本较低的运输方式
2. **蒙特卡洛模拟**：10,000次模拟进行敏感性分析和风险评估
3. **水资源需求模型**：基于ISS生命维持系统数据计算
4. **环境影响评估**：碳排放、综合环境指数、环境评分

## 建模创新点

1. **技术进步建模**：考虑火箭发射频率增长（5%/年）和成本下降（5%/年）
2. **分层调度策略**：优先使用成本低的太空电梯，火箭补充剩余需求
3. **环境影响量化**：建立综合环境指数和环境评分系统
4. **动态调整机制**：逐年优化运输分配
5. **数据验证**：完善的数据验证和合理性检查

## 主要发现

1. **技术进步的关键作用**：考虑技术进步后，火箭方案从3024年缩短至90年
2. **混合方案的优势**：35年完成，是唯一可在合理时间内完成建设的方案
3. **环境效益显著**：混合方案碳排放仅为纯火箭方案的9%
4. **成本可控**：总成本435亿美元，远低于纯太空电梯方案（5940亿美元）
5. **鲁棒性好**：95%置信区间内方案稳定可行
