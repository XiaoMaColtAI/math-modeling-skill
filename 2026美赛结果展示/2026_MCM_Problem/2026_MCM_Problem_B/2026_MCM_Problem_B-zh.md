# 问题 B：利用太空电梯系统建造月球殖民地

![](images/d56c8f8a71b712570aacdb699e2e19450e8b1a708748d394f92cbee59747b814.jpg)


想象一个未来，任何人都可以通过从赤道到地球轨道的悠闲观光之旅进入太空，然后乘坐常规、安全且廉价的火箭航班前往月球、火星或更远的地方。在这个未来，我们可以建造郁郁葱葱、绿色美丽的太空栖息地，配备人工重力，人们可以在那里度假、工作甚至生活。这些栖息地将减轻地球脆弱、过度负荷和敏感生态系统的压力。实现这些目标的技术将为人类提供无限、安全、常规、环保、高效且全球通行的太空通道。为了实现这些目标，一些人设想了一个由电力驱动的太空电梯系统，为行星际物流、商业和探索提供可扩展的基础设施。

在最终运行配置中，太空电梯系统将由三个银河港组成，理想情况下在赤道周围相隔120度。每个银河港将包括一个地球港口，连接着两个10万公里长的系绳，分别连接到两个顶端锚点，多个太空电梯协同运行，每个电梯每天都能将大量有效载荷从地球提升到地球同步轨道（GEO）以及更远的顶端锚点，在那里它们可以装载到火箭上，使用更少的燃料运送到任何地方。

月球殖民管理局（MCM）计划在太空电梯系统完成后，于2050年开始建造一个可容纳约10万人的月球殖民地。据估计，月球殖民地将需要约1亿公吨的材料。此外，一旦殖民地建成，还需要定期向月球输送水和物资以维持其人口生存。要到达月球，银河港必须分两步运送材料：首先，通过太空电梯从地球港口到顶端锚点；其次，通过火箭从顶端锚点到月球殖民地。MCM局预计银河港将提供一个先进的提升系统，每年能够运送17.9万公吨物资，同时不产生大气污染。

该局也在考虑使用传统火箭为月球殖民地提供建筑材料和物资。地球目前有十个火箭发射场：阿拉斯加、加利福尼亚、得克萨斯、佛罗里达和弗吉尼亚（美国）、哈萨克斯坦、法属圭亚那、萨蒂什·达万航天中心（印度）、太原卫星发射中心（中国）和玛希亚半岛（新西兰）。

火箭只需从地球上的火箭发射场一步即可到达月球殖民地。据估计，到2050年，火箭将能够使用先进的猎鹰重型发射任务将100-150公吨的有效载荷运送到月球。您可以假设银河港系统（例如系绳不摇摆）和火箭发射（例如发射不失败）都处于完美条件。您应考虑在不同场景下将材料从地球表面运送到月球殖民地现场的成本和时间表。

# 您的任务：

您的任务是利用数学模型来确定从2050年开始运输材料建造一个可容纳10万人的月球殖民地的成本和相关时间表。您需要比较现代太空电梯系统的三个银河港与从选定火箭基地发射的传统火箭。

# 术语表

太空电梯系统 由三个银河港以及额外支持设施组成。

银河港 由两个顶端锚点组成，每个锚点通过两条系绳连接到单个地球港口。

地球港口 是地球上为银河港提供表面支持的位置。

系绳 是长10万公里的石墨烯材料，连接太空电梯系统中的地球港口和顶端锚点。

顶端锚点 是太空中10万公里系绳末端处的配重。

地球同步轨道（GEO） 是地球表面以上约35,786公里的轨道，绕地球运行的轨道周期为24小时，与地球自转相匹配，因此每天保持在相同的经度上。

月球殖民地 是月球上可容纳10万人的栖息地。
