# 术语表 (Terminology)

本文档用于确保数学建模全过程的术语一致性，便于论文撰写和英文化。

## 一、核心概念术语

| 中文名称 | 英文名称 | 缩写符号 | 定义/说明 |
|---------|---------|---------|----------|
| 荷电状态 | State of Charge | SOC | 电池当前剩余容量与额定容量的比值，取值范围[0,1] |
| 时间耗尽 | Time-to-Empty | TTE | 电池从当前状态完全耗尽所需的估计时间（小时） |
| 功率消耗 | Power Consumption | P | 单位时间内消耗的电能（瓦特，W） |
| 电池容量 | Battery Capacity | Q | 电池能够存储的电荷量（毫安时，mAh） |
| 额定容量 | Nominal Capacity | Q_nom | 电池设计时的标称容量 |
| 有效容量 | Effective Capacity | Q_eff | 考虑老化后的实际可用容量 |
| 截止荷电状态 | Cutoff State of Charge | SOC_cutoff | 电池保护的最低荷电状态（通常为5%） |
| 端电压 | Terminal Voltage | V | 电池正负极之间的电压（伏特，V） |
| 标称电压 | Nominal Voltage | V_nom | 电池的标称工作电压 |
| 内阻 | Internal Resistance | R_int | 电池内部的等效电阻（欧姆，Ω） |
| 开路电压 | Open Circuit Voltage | OCV | 电池无负载时的电压 |
| 等效电路模型 | Equivalent Circuit Model | ECM | 用电路元件模拟电池行为的数学模型 |
| 能量密度 | Energy Density | - | 单位质量或体积的电池储存的能量 |
| 功率密度 | Power Density | - | 单位质量或体积的电池输出功率 |

## 八、评价指标术语

| 中文指标 | 英文指标 | 公式/定义 |
|---------|---------|----------|
| 均方根误差 | Root Mean Square Error | RMSE = $\sqrt{\frac{1}{n}\sum_{i=1}^n(y_i-\hat{y}_i)^2}$ |
| 平均绝对百分比误差 | Mean Absolute Percentage Error | MAPE = $\frac{100\%}{n}\sum_{i=1}^n|\frac{y_i-\hat{y}_i}{y_i}|$ |
| 决定系数 | Coefficient of Determination | $R^2 = 1 - \frac{SS_{res}}{SS_{tot}}$ |
| 平均绝对误差 | Mean Absolute Error | MAE = $\frac{1}{n}\sum_{i=1}^n|y_i-\hat{y}_i|$ |
| 预测区间覆盖率 | Prediction Interval Coverage | PIC = 实际值落在预测区间的比例 |
| Sobol敏感性指数 | Sobol Sensitivity Index | 衡量参数对输出方差的贡献 |

## 使用说明

1. **术语一致性**：在建模、编程、论文各阶段使用统一的术语
2. **符号规范**：数学符号应与本文档保持一致
3. **单位统一**：推荐使用国际单位制（SI），特殊情况注明
4. **英文翻译**：撰写英文论文时参考本文档的英文名称
5. **扩展原则**：新增术语时按本文档格式补充
