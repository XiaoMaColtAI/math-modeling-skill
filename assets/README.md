## 使用建议

### 对于建模手
1. 根据**题型特征**快速定位合适的算法类别
2. 查阅算法的**适用范围**表，确认匹配度
3. 参考**关键文献**获取理论支撑
4. 记录选定的算法和公式供论文使用

### 对于编程手
1. 参考**代码实现要点**进行代码编写
2. 根据推荐的**可视化图表类型**绘制结果
3. 注意代码中的参数设置和边界条件处理

### 对于论文手
1. 使用文档中的**数学公式**描述模型
2. 参考**关键文献**编写参考文献
3. 说明**算法选择理由**（使用适用范围部分）
4. 结合**可视化图表**展示结果
